# PHP MySQL Country Service

A PHP-based web service that retrieves country and city information from a MySQL database
and serves results via AJAX requests.

## Features
- PHP backend integrated with MySQL
- SQL queries including JOIN operations
- AJAX-based frontend for dynamic search
- Supports partial searches and full listings
- Displays structured results in HTML tables

## Tech Stack
- PHP
- MySQL
- JavaScript (AJAX)
- HTML
- CSS

## How to Run
1. Import the provided SQL file into MySQL using phpMyAdmin.
2. Update database credentials in `world.php`.
3. Place project in `htdocs` or `www` directory.
4. Start Apache and MySQL servers.
5. Open `index.html` in your browser.
